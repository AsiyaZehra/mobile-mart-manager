# Mobile Mart Manager

## Overview
A desktop-based management system built to streamline operations 
for a mobile retail store — covering inventory, sales, and customer management.

## Tech Stack
- Java / J2SE
- MySQL
- JDBC

## Features
- 📦 Inventory management — track stock levels and product details
- 💰 Sales tracking — record and monitor transactions
- 👤 Customer management — maintain customer records
- 📊 Reporting — generate sales and inventory summaries

## What I Learned
- Built end-to-end application using Java and MySQL
- Designed relational database schema for retail operations
- Wrote technical documentation for long-term maintainability
- Followed Agile development practices in a cross-functional team
